css居中问题

发表于 2016-07-05   |  

  • 经常遇见文字图片不能居中对齐的问题，在css中设置图片的vertical-align属性就可以了：

             #img{
             vertical-align:middle;
         }

  • css3垂直居中几种思路：

    1.利用position和负边距

    先设置容器的宽高，给他父级设置position: relative;

    css部分是这样：

      .selector {
      position: absolute;
      top:50%;
      margin-top:-元素自身高度的一半;(是负边距，前面有个负号）
  }

    2.table-cell

    结构：
<div class="div1">
       <span>让我居中试试</span>
</div>

样式：
.box1{
        display: table-cell;
    vertical-align: middle;
    text-align: center;
}

    3.display:flex (弹性布局）

    .box{
    display: flex;
    justify-content:center;
    align-items:Center;
}

    4.translate变移位实现居中

    .div span{
    position: absolute;
    top:50%;
    left:50%;
    width:100%;
    transform:translate(-50%,-50%);
    text-align: center;
  }

    5.display:inline-block

    .div{
     text-align:center;
     font-size:0;
}
.div span{
     vertical-align:middle;
     display:inline-block;
     font-size:16px;
}
.div:after{    /*使用after这个伪类来撑开容器，这种想法值得借鉴，非常巧妙*/
     content:'';            
     width:0;
     height:100%;
     display:inline-block;
     vertical-align:middle;
}
