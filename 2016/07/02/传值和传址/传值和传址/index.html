传值和传址

发表于 2016-07-02   |  

在js中，对于简单的数据类型(number,string,boolean),函数是按值传递的，在函数中有操作对实参没有影响，但是如果是复杂的数据类型(如数组，对象)，那么传递过程中传的是指针，也就是说引用了某个地址下面的数据，那么函数的操作会影响被引用的对象(本人的理解，看不懂勿怪)，例如：

var arr1 = [1,2,3,4];
var arr2 = arr1;

arr2[0] = 10;
console.log(arr1[0]);

可以看到console.log打印出arr1[0]的值变成了10，在这里我们并没有对arr1有任何操作，只是将arr2数组的第一个值改成了10，结果影响了arr1的值，这就是因为arr2的地址空间和arr1的地址空间是一样的，修改arr2的内容等于修改了这个地址空间内的内容，所有引用这个地址空间的数据内容都会变化，传址实际上就是传递这个地址空间的地址，指针就是指向这个地址空间。

而传值就是复制了一个地址空间的内容存在另一个地址空间，二者互相没有干扰所以修改一个地址空间的内容不会影响另一个地址空间(应该可以这么理解)。
