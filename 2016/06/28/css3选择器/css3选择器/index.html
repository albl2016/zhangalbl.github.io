css3选择器

发表于 2016-06-28

子元素选择器 vs后代选择器

格式：E > F VS E F

子元素选择器选择的是E的子元素F,不包括F的后代元素，而后代选择器选择的是E后代F，包括子子孙孙。

    .div1 > h2{            /* 表示选择div1的子元素的h2,对其设置一个1像素的红色边框 */
        border: 2px red solid;
    }
    .div1 h2{               /* 表示选择div1下面所有的后代子孙的h2,设置背景为红色*/
        background: red;
    }

兄弟选择器

分为：相邻兄弟选择器 vs 通用兄弟选择器

格式： E + F VS E ~ F

相邻兄弟选择器（父级相同）选择的必须是同级的、相邻的（重点是相邻元素）、在它下方（不是它的子元素，是同级的从上到下排列，位置在上方也不行）的元素。例如：

    div + h2{
        border: 1px blue solid;            /* 选择div下方和它自己相邻而且同级的h2，对不相邻的不会选择，只有一个*/
    }

    div ~ h2{
        border: 2px red solid;            /* 选择div下方和它同级的的元素h2，对不相邻的也会选择，可以选择多个 */
    }

属性选择器

格式：E[attr] 例如：E[class]表示选择属性名为class的元素E

E[attr=“value”] 例如：E[class=”div1”]表示选择属性名为class且属性值为”div1”的元素E

E[attr~=“value”] 例如：E[class~=”div1”]表示选择几个以空格隔开的属性值中包含了”div1”的元素E

E[attr ^ =“value”] 例如：E[class^=”div1”]表示选择以属性值为”div1”开头的元素E

E[attr $ =“value”] 例如：E[class$=”div1”]表示选择以属性值为”div1”结尾的元素E
E[attr =“value”] 例如：E[class=”div1”]表示选择属性值包含了”div1”的元素E

E[attr|=“value”] 例如：E[class|=”div1”]表示选择以属性值为”div1”或”div1-“开头的元素E

例如我们要实现将超链接变成红色，可以这样写:

a[href="http://www.baidu.com"] {color: red;}

伪类选择器

伪类选择器主要分为锚点伪类和UI元素伪类（针对表单元素）。

锚点伪类主要是link，visited，hover，active和focus.它们表达的意思是：

:link 元素没有被访问的状态。

:visited 元素被访问过后的状态。

:hover 鼠标悬浮在元素上的状态。

:active 点击元素那一下的状态。

:focus 表示元素成为焦点的状态。

例如： input:focus{border:10px red solid;} 运行结果如下图：

UI元素伪类主要有:enabled，:disabled，:checked，这些主要是对HTML中form元素操作.

在input元素中，type=”text”有两种不同的状态分别是disabled(不可编辑状态)和enabled(可编辑状态).type=”checkbox”也有checked和unchecked两种状态。例如：

      
 在css中：
        input[type="text"]:disabled{border: 4px red solid;}

nth选择器

:first-child 选择某个元素的第一个子元素

:last-child 选择某个元素的最后一个子元素

:nth-last-child()选择某个元素的一个或多个特定的子元素，从这个元素的最后一个子元素开始算；

:nth-of-type()选择指定的元素；

:nth-last-of-type()选择指定的元素，从元素的最后一个开始计算；

:first-of-type选择一个上级元素下的第一个同类子元素；

:last-of-type选择一个上级元素的最后一个同类子元素；

:only-child选择的元素是它的父元素的唯一一个子元素；

:only-of-type选择一个元素是它的上级元素的唯一一个相同类型的子元素；

:empty选择的元素里面没有任何内容。

:nth-child()选择某个元素的一个或多个特定的子元素；

简单的说，我们可以是用nth选择器对这个ul中的li设置不同的样式，:nth-child()可以用来设置特定的某一行的样式，(odd表示奇数行，even偶数行)例如：

ul li:nth-child(5){
    width: 200px;                 
    background: yellow;      
}                                      /* 设置第五行背景色为黄色 */

ul li:nth-child(odd){                
    width: 200px;                  
    background: red;
}                                     /* 设置奇数行背景色为红色 */

ul li:nth-child(even){                
    width: 200px;                      
    background: blue;
}                                    /* 设置偶数行背景色为蓝色 */

隔行变色运行结果如下图：

ul li:nth-child(2n+1){
    width:200px;
    background: grey;       
}                                    /* 设置隔行变色的样式 */

ul li:nth-child(-n+3){
    width:200px;
    background: green;
}                                    /* 选择小于3后面的列表行设置为绿色 */
