html5语义化标签--表单元素和属性篇

发表于 2016-06-28   |  
表单元素和属性篇

表单元素

fieldset

legend

samp 定义计算机程序的样本文本

ins

del 定义已经被插入文档中的文本

kbd（已废弃）

keygen

datalist 配合input使用，list属性配合id名

label
for
input

color 用于通过颜色池控件选择颜色。

range 在大部分支持该类型的浏览器中，范围控件的实施形式为滑块。用于输入数字，但与 number 的区别在于无需输入具体数字。

number 用户输入数字

datetime-local
用于输入不含时区的日期和时间。

time
用于输入含小时、分钟、秒和秒的小数部分但不含时区的时间值。

week
例如，此格式的日期 2011-W05 表示 2011 年的第 5 周。用于输入含年份和周数但不含时区的日期。

month
用于输入含年份和月份但不含时区的日期。

date
用于输入不含时区的日期。

datetime
用于输入时区设置为 UTC 的日期和时间。

email
如果指定了 multiple 属性，就可以输入多个电子邮件地址，以逗号分隔，用于输入单个电子邮件地址

url 用于输入单个网址。

search search 和 text 之间的区别主要在于样式上用于提示用户输入要搜索的文字。

tel 用于输入电话号码。

表单属性

form

autocomplete 自动补全

pattern 高级验证

required

placeholder 在对焦到相关元素以及用户输入数据之前，系统会以浅色文字显示占位符值。您可以在输入和文本区域中指定该值，用于提示用户应输入的数据类型。

<form action="">
    <input type="text" placeholder="请输入你的帐号" autocomplete="off"/>
    <input type="password" placeholder="请输入您的密码"/>
    <input type="submit"/>
</form>
